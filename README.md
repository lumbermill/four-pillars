# four-pillars
四柱推命


## コマンドラインで試す

```
ruby railsapp/lib/tasks/four-pillars.rb -h
```

## 開発用のイメージを立ち上げる
[Docker Desktop on Mac](https://docs.docker.com/docker-for-mac/install/) をインストールしてください。

```
docker-compose up
```

## Herokuにデプロイする

```
git subtree push --prefix railsapp heroku master
```
https://four-pillars.herokuapp.com/
